# FastFuriorTransform
A 2-demensional fast furior transform project for image processing.

数字图像处理课作业，程序由C++语言编写，用到的库如下：

#include <stdio.h>
#include <math.h>
#include <cv.h>
#include <highgui.h>
#include <cvaux.h>
#include <Windows.h>
#include <opencv2/opencv.hpp>
#include <opencv2/imgproc/imgproc.hpp>
